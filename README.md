# examen-final-computacion-visual-jorge-alandete.
Repository of the final exam of computer vision course.

## Resumen del Proyecto

Este repositorio reúne dos ejercicios prácticos de Computación Visual, abordando tanto el procesamiento digital de imágenes como la interacción en entornos 3D.

## Parte 1 – Procesamiento Visual con Python y YOLO

Se desarrolló una aplicación capaz de procesar imágenes y videos aplicando diferentes transformaciones visuales, incluyendo conversión a escala de grises, espacios de color HSV y LAB, detección de bordes mediante Canny y detección de objetos utilizando YOLOv8. El objetivo fue combinar técnicas clásicas de visión por computador con modelos modernos de inteligencia artificial para analizar y resaltar distintas características de la información visual.

## Parte 2 – Escena 3D Interactiva en Unity

Se implementó una simulación de una línea de producción industrial en Unity donde el usuario controla un brazo robótico encargado de sellar cajas transportadas por una cinta. El proyecto integra conceptos de modelado jerárquico, animación, detección de colisiones, raycasts e interacción en tiempo real, recreando un flujo básico de automatización industrial.

## Tecnologías Utilizadas
- Python
- OpenCV
- NumPy
- YOLOv8 (Ultralytics)
- Unity Engine
- C#

## Resultados Generales

Los ejercicios permitieron aplicar conceptos fundamentales de visión por computador, procesamiento digital de imágenes, inteligencia artificial, simulación 3D e interacción en tiempo real, demostrando aplicaciones prácticas tanto en análisis visual como en automatización de procesos.

Para información detallada sobre la arquitectura, implementación, instalación, ejecución y resultados de cada ejercicio, consulte los README específicos de cada proyecto.
